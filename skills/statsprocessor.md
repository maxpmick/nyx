# statsprocessor

## Overview
Statsprocessor (sp) is a high-performance word generator that uses Markov chains to determine the probability of character sequences. It reads a `.hcstat` file (generated from an analysis of existing dictionaries) to produce candidate passwords where each letter's position is influenced by the preceding letter. Category: Password Attacks.

## Installation (if not already installed)
The tool is typically pre-installed on Kali Linux. If missing:

```bash
sudo apt install statsprocessor
```

## Common Workflows

### Basic Markov-based generation
Generate words based on a statistics file with default settings:
```bash
statsprocessor /usr/share/hashcat/hashcat.hcstat
```

### Length-constrained generation
Generate words between 6 and 8 characters long:
```bash
statsprocessor --pw-min=6 --pw-max=8 /usr/share/hashcat/hashcat.hcstat
```

### Using a filter mask
Generate 8-character words starting with a capital letter, followed by lowercase letters, based on Markov stats:
```bash
statsprocessor /usr/share/hashcat/hashcat.hcstat ?u?l?l?l?l?l?l?l
```

### Distributed processing
Skip the first 1,000,000 words and limit output to the next 500,000 words:
```bash
statsprocessor -s 1000000 -l 500000 /usr/share/hashcat/hashcat.hcstat
```

## Complete Command Reference

The tool is available as `statsprocessor`, `sp32`, or `sp64`.

```
statsprocessor [options]... hcstat-file [filter-mask]
```

### Startup Options
| Flag | Description |
|------|-------------|
| `-V`, `--version` | Print version |
| `-h`, `--help` | Print help |

### Increment Options
| Flag | Description |
|------|-------------|
| `--pw-min=NUM` | Start incrementing at NUM |
| `--pw-max=NUM` | Stop incrementing at NUM |

### Markov Options
| Flag | Description |
|------|-------------|
| `--markov-disable` | Emulates maskprocessor output (disables Markov logic) |
| `--markov-classic` | No per-position tables |
| `--threshold=NUM` | Filter out chars after NUM chars added (Set to 0 to disable) |

### Misc Options
| Flag | Description |
|------|-------------|
| `--combinations` | Calculate and display the number of combinations |
| `--hex-charset` | Assume charset is given in hex |

### Resource Options
| Flag | Description |
|------|-------------|
| `-s`, `--skip=NUM` | Skip number of words (useful for restoring sessions) |
| `-l`, `--limit=NUM` | Limit number of words (useful for distributed attacks) |

### Files Options
| Flag | Description |
|------|-------------|
| `-o`, `--output-file=FILE` | Write results to specified file |

### Custom Charsets
| Flag | Description |
|------|-------------|
| `-1`, `--custom-charset1=CS` | User-definable charset 1 (e.g., `--custom-charset1=?dabcdef`) |
| `-2`, `--custom-charset2=CS` | User-definable charset 2 |
| `-3`, `--custom-charset3=CS` | User-definable charset 3 |
| `-4`, `--custom-charset4=CS` | User-definable charset 4 |

### Built-in Charset Placeholders
| Placeholder | Description |
|-------------|-------------|
| `?l` | abcdefghijklmnopqrstuvwxyz |
| `?u` | ABCDEFGHIJKLMNOPQRSTUVWXYZ |
| `?d` | 0123456789 |
| `?s` | ` !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~` |
| `?a` | `?l?u?d?s` |
| `?h` | 8 bit characters from 0xc0 - 0xff |
| `?D` | 8 bit characters from German alphabet |
| `?F` | 8 bit characters from French alphabet |
| `?R` | 8 bit characters from Russian alphabet |

## Notes
- The `.hcstat` file is a binary file containing character frequency statistics. Standard files are often found in `/usr/share/hashcat/` or `/usr/share/oclhashcat/`.
- Using `--threshold` can significantly reduce the size of the generated wordlist by excluding low-probability character transitions.
- When piping to another tool, ensure the receiving tool can handle the high-speed stream generated by `statsprocessor`.
